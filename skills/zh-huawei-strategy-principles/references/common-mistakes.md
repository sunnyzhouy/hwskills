# Common Mistakes

Use this file when the user has a polished draft that still feels strategically thin.

## Common mistakes

- Replace choices with broad aspirations
- Skip capability diagnosis and jump straight to actions
- Write the annual plan as disconnected departmental tasks
- Ignore what must stop in order to focus
- Use lagging metrics only and omit control points

## Correction pattern

1. Restate the business ambition.
2. Name the main external and internal gaps.
3. Force the draft to choose.
4. Rebuild the annual plan from those choices.
5. Add owners, milestones, and metrics.
