# Workflow

Use this file when the user needs the full lead-to-cash flow.

## Full sequence

1. Define the process boundary from lead to cash.
2. Clarify stage exits and review cadence.
3. Map cross-functional handoffs across sales, delivery, and finance.
4. Define control points for contract, acceptance, and collection.
5. Set metrics and escalation logic.

## Review prompts

- What makes a lead qualified enough to enter the pipeline?
- What must be true to move a deal to commitment?
- Where do deals get stuck between sales and delivery?
- What causes revenue without cash realization?
